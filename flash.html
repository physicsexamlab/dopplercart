Doppler Cart ファームウェア書き込み

Arduino Nano ESP32 に Doppler Cart のファームウェアを書き込みます。
書き込みには Google Chrome（または Edge）が必要です。

  1. Arduino Nano ESP32 を USB ケーブルで PC に接続する
  2. 下の「書き込む」ボタンをクリックする
  3. 表示されるポート選択ダイアログで該当ポートを選択する
  4. 書き込みが完了したら USB を抜き差しして再起動する
  5. 各基板に固有名のラベルを貼る（下記「デバイス名」参照）
Arduino Nano ESP32 (ESP32-S3)

※ ブラウザが「シリアルポートへのアクセス」の許可を求めます。「許可」を選択してください。
※ 書き込み中はボードを抜かないでください。
※ Safari・Firefox には Web Serial API が実装されていないため使用できません。

デバイス名について
各基板は Doppler IQ Sensor XXXX という名前で BLE 広告します。 XXXX は基板固有の4桁（Bluetooth MAC 末尾）で、再起動しても変わりません。 同一教室内で複数台を使う場合、この名前で各台を区別できます。
名前の確認方法: 書き込み後、シリアルモニタ（115200 bps）に BLE started. name="Doppler IQ Sensor XXXX" ... と表示されます。 この4桁を各カートにラベルで貼っておくと、生徒が接続ダイアログで自分のカートを選べます。
